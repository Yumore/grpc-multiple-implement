# gRPC Chat Example with Node JS

This project creates a simple command line chat using gRPC and NodeJS to show a stream and bidirectional communication.

- Install Modules:

    ``$ npm install``

- Start Server:

    ``$ node server``

- Start client:

    ``$ node client``
